## 📖 Description

**Educational News Website** is a simple learning project created using **Django**, **PostgreSQL**, and **Tailwind CSS**.

The project demonstrates basic **CRUD operations** using **Django ORM** with a clean and minimal frontend.

> ⚠️ This project is created for educational purposes only.

### Limitations
- ❌ No authentication
- ❌ No user roles
- ❌ Minimal functionality
- ✅ Focus on backend fundamentals
